# Cinema Reservation API

A FastAPI-based cinema reservation backend with authentication, bookings, screenings, and movie management.

## Features
- FastAPI application structure
- JWT-based authentication
- Async PostgreSQL support
- Celery and Redis task support
- Alembic migrations

## Setup
1. Copy .env.example to .env and adjust the values.
2. Build and start the services with Docker Compose.
3. Run the API and migrations as needed.
